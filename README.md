## 🎮 How to Update Games
Edit **`js/games.js`** only — change `playLink`, `downloadLink`, and `image` for each game.

## 🚀 GitHub Pages Deployment
1. Push all files to your GitHub repo
2. Go to **Settings → Pages → Source → main branch → / (root)**
3. Add `gamesplatform.com` as custom domain in Namecheap DNS
